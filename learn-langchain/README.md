## what is it ?
- LangChain is open source framework for building LLM powered application
- it has

1. Chain - One component output became input for another component
2. Model Agnostic Development - almost any model can be used just focus on core problem 
3. Complete Eco System - Many tool / utilities for solving common problem 
4. Memory and state handling - know the context

### what can build with it
- ChatBot
- AI agents work behalf on us 
- Workflow Automation
- Summarization 

- [llamaindex](https://www.llamaindex.ai/) and [haystack](https://haystack.deepset.ai/) are alternative example of langchain
